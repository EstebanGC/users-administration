# 📝 Users Administration App

Application to handle contacts, users and phone. Coded with NestJS using Typeorm and Docker to run PostgreSQL virtually. 

## ⚙ Installation

```bash
$ npm install
```

## 🛠 Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
